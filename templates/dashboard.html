Integrazioni

MCP Server — client LLM esterni

SentinelNet è utilizzabile da client LLM esterni (Claude Desktop, LM Studio, Cline…) tramite il protocollo MCP (Model Context Protocol): il ponte mcp_server.py gira sulla macchina del client, si autentica con un account SentinelNet e inoltra le richieste all'API REST. Ruoli, tenant e blacklist comandi restano applicati lato server. Consiglio: crea un account dedicato (viewer per sola lettura, operator se servono i comandi CLI).

Configurazione client

Snippet per claude_desktop_config.json (o equivalente del tuo client). Sostituisci utente e password con l'account dedicato; il percorso deve puntare alla copia locale di mcp_server.py.
